# @vmd/mobile — VMDHub Mobile Application

React Native app for iOS and Android with local SQLite storage.

## Development

```bash
npm start            # Start Metro bundler
npm run android      # Run on Android emulator/device
npm run ios          # Run on iOS simulator (macOS only)
```

## Build

```bash
npm run build:android   # Android .apk
npm run build:ios       # iOS archive
```

## Screens

- **Home** — Dashboard with monthly stats
- **Invoices** — Invoice list
- **Vendors** — Vendor directory
- **Reports** — Monthly revenue charts
- **Settings** — Company info, currency
